# Port Authority Management System

A full-stack system to manage port operations including ships, docks, cargo, and container tracking.

## Tech Stack
- Frontend: React (Vite + Tailwind)
- Backend: Node.js (Express)
- Database: MongoDB

## Core Modules
- Ships
- Dock Management
- Cargo Handling
- Container Tracking

## Key Feature
Real-time logistics simulation:
Ship → Dock → Cargo → Container Movement
